Анализ открытия кофейни Surf Coffee рядом с ВШЭ на Покровском бульваре


Бизнес-идея

Компания открывает кофейню **Surf Coffee** на Покровском бульваре рядом с корпусом ВШЭ. Основной продукт — кофе и напитки высокого качества, а также разнообразная еда: выпечка, десерты и лёгкие блюда. Кофейня делает акцент на атмосфере, бренде и качестве продукта.

Целевая аудитория:студенты и сотрудники ВШЭ — люди, которым важны удобство, вкус, эстетика пространства и возможность купить кофе рядом с корпусом.

---

Исследовательский вопрос

> Есть ли смысл открывать кофейню Surf Coffee рядом с корпусом ВШЭ на Покровском бульваре — кто будет целевой аудиторией, какие конкуренты уже есть рядом и за счёт чего новая кофейня может конкурировать?

Методы

- Ручной сбор: список конкурентов из Яндекс Карт
- ООП-парсинг: класс `CoffeeSiteParser` (тема курса ✓)
- Регулярные выражения: извлечение цен и объёмов из HTML
- Pandas: обработка и очистка данных
- Matplotlib: визуализация результатов

Промежуточные выводы

- В ближайшем окружении Покровского бульвара работают 7 конкурентов разных форматов.
- Сетевые конкуренты: Jeffrey's Coffee и Правда кофе — имеют узнаваемый бренд и стандарты.
- Локальные конкуренты: Glitch Coffee, 2Grind, Corner Coffee — делают ставку на атмосферу и качество кофе.
- Правда кофе работает в формате кофе to go с акцентом на скорость, Surf Coffee может предложить больше — атмосферу и пространство.
- Анализ цен Правда кофе показывает рыночный диапазон: ~170–280 руб. за основные напитки.
- Потенциальная ниша Surf Coffee: узнаваемый бренд + пространство для учёбы/работы + ориентация на университетскую аудиторию.
