# Анализ открытия кофейни Surf Coffee рядом с ВШЭ на Покровском бульваре

**Курс:** Наука о данных, spring 2026
**Раздел исследования:** Анализ конкурентов + целевой аудитории (1 и 2 из примеров в ТЗ)

---

## Бизнес-идея

Открытие кофейни **Surf Coffee** на Покровском бульваре рядом с корпусом ВШЭ.
Основной продукт — кофе высокого качества и еда: выпечка, десерты, лёгкие блюда.
Surf Coffee — известный федеральный бренд с собственной эстетикой и культурой, ценимый среди молодой аудитории и студентов.

**Целевая аудитория:** студенты и сотрудники ВШЭ.

## Ключевые результаты

### Конкуренты

В радиусе 500 м от ВШЭ — 7 кофеен. Прямой конкурент по формату — **Jeffrey's Coffee** (тот же адрес, Покровский 11). Средний рейтинг конкурентов на Яндекс.Картах — 4.6. Рынок зрелый, но не насыщен новыми брендированными форматами.

### Опрос (n = 156)

- **88%** покупают кофе вне дома, **60%** — минимум 3 раза в неделю.
- Средний чек: **234 ₽** (95% CI: 224–245 ₽).
- Топ-3 фактора выбора: расположение, вкус, цена.
- Топ-3 напитка: латте, капучино, американо.
- **Узнаваемость Surf Coffee — 80%**, выше всех конкурентов. Это главное преимущество.
- t-test: сотрудники тратят значимо больше студентов (p < 0.0001).

### Финансовая модель

| Метрика | Значение |
|---------|----------|
| Средний чек | 270 ₽ |
| Базовый трафик | 240 чеков/день |
| Выручка/мес | 1.94 млн ₽ |
| Операционная прибыль/мес | 344 000 ₽ |
| Операционная маржа | 17.7% |
| Точка безубыточности | 173 чека/день |
| Стартовые инвестиции | 4.5 млн ₽ |
| Окупаемость | ~13 месяцев |

### Концепция

Surf Coffee у ВШЭ — кофейня с известным федеральным брендом, ориентированная на студентов. Главное преимущество — узнаваемость и доверие к бренду, которое уже есть. Подробнее в `CONCEPT.md`.
